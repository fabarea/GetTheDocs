TYPO3 Documentation Service

Documentation of the TYPO3 is written in reStructuredText format. The reStructuredText (frequently abbreviated as reST) is a lightweight markup language intended to be highly readable in source format. By the means of processors, reST documentation can be rendered into various formats for printing or screen, such as PDF, ePub, HTML...

Command line tool

In a "zero config" effort, the Doc Team is providing a command line tool aiming to render documentation on-line without the need to install the whole build chain.

Installation

Download the script somewhere in your system. On a *nix system, it can be achieved from the command line with: 

	cd PATH
	curl -s http://preview.docs.typo3.org/getthedocs/get-the-docs.php > get-the-docs;
	chmod 755 get-the-docs

Tip: putting the script somewhere in your PATH (e.g /usr/bin/) would make the command recognized by the system.

Requirement

the script requires PHP 5.2 and above with Zip support enabled which is often provided by package "php5-zip". To verify its support, type in a terminal. 

	php -i | grep zip

Usage

	$ get-the-docs help

	Toolbox for managing TYPO3 documentation

	Usage:
		get-the-docs render     Render documentation remotely
		get-the-docs convert    Convert legacy OpenOffice documentation to reST
		get-the-docs config     Download configuration for local rendering (Working but still missing features...)
		get-the-docs help       Print this help

Local installation

It is also possible to install the "build chain" on your local machine for speed purposes or because of more advanced rendering. Refer to our wiki to get started. In a nutshell, you will find resources to

  • install Sphinx
  • install the TYPO3 theme
  • generate and configure build files
