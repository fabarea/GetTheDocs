TYPO3 Documentation Services

Documentation of the TYPO3 is written in reStructuredText format. The reStructuredText (frequently abbreviated as reST) is a lightweight markup language intended to be highly readable in source format. By the means of processors, reST documentation can be rendered into various formats for printing or screen, such as PDF, ePub, HTML...

Command line tool

In a "zero config" effort, the Doc Team is providing a command line tool aiming to render documentation on-line without the need to install and configure the whole build chain. To download and set up the tool, refer to this page.

At the end of the installation, read the help from the command directly: 

	$ get-the-docs help
You will be able to generate the docs in one command. 
	$ get-the-docs render PATH/TO/EXTENSION

Local installation

It is also possible to install the "build chain" on your local machine. Rendering on a local machine is fast and allow custom transformation. Refer to our wiki to get started. In a nutshell, you will need to go through theses steps:

  • install Sphinx
  • install the TYPO3 theme
  • generate and configure build files
