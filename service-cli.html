Command line tool

This script will help you in generating TYPO3 documentation from the command line.

Installation

Download the script somewhere in your system. On a *nix system, it can be achieved from the command line with: 

	cd PATH
	curl -s http://preview.docs.typo3.org/getthedocs/get-the-docs.php > get-the-docs;
	chmod 755 get-the-docs

Tip: putting the script somewhere in your PATH (e.g /usr/bin/) would make the command recognized by the system.

Requirement

the script requires PHP 5.2 and above with Zip support enabled which is often provided by package "php5-zip". To verify its support, type in a terminal. 

	php -i | grep zip

Usage

	$ get-the-docs help

	Toolbox for managing TYPO3 documentation

	Usage:
		get-the-docs render     Render documentation remotely
		get-the-docs convert    Convert legacy OpenOffice documentation to reST
		get-the-docs config     Download configuration for local rendering (Working but still missing features...)
		get-the-docs help       Print this help
